Boxcript(박스크립트)에 대한 소개

Boxcript는 중앙전산소 병장 심영재(Mr. 하늘)가 스크립트 언어가 어떻게 개발되는지 궁금해
3일동안 만든 간단한 프로그래밍 언어입니다.

특징은 다음과 같습니다.
1. 모든것이 객체 (test/object.box를 참고하세요.)
2. 연산자 정의 (test/operator.box를 참고하세요.)
3. 자바와 연동 (test/java.box를 참고하세요.)
4. 다중 상속 (test/import.box를 참고하세요.)

※ 실행 방법
콘솔 창에서 java -jar <boxcript.jar의 위치> <실행할 파일의 위치>

※ 주의 사항
실행할 파일의 인코딩은 UTF-8로 하시기 바랍니다.
코드 Cache 기능이 구현되지 않아 실행 속도가 굉장히 느립니다.
